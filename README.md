Name: *InspecTask* - Task Management Application

Core Features
1. Task Input Form
. Add tasks with an input field and a submit button.
. Ensure the input is validated to prevent empty or duplicate tasks.
2. Task List
. Display tasks dynamically, showing task names and their completion
status.
. Allow users to mark tasks as complete or delete them.
3. Persistent Data
. Use localStorage to save tasks so they persist even after refreshing
the page.
4. Progress Tracker
. Implement a visual progress tracker that shows the percentage of
tasks completed.
. Update the progress dynamically as tasks are marked as complete or
pending.
5. Task History
. Allow users to view a history of completed tasks.
. Provide an option to restore or delete tasks from the history, giving
users control over their task management.

Technical Requirements:
. Frontend Framework: React.js
. Data Storage: localStorage
. Version Control: Git and GitHub for version tracking
. Deployment: Deploy on GitHub Pages or Netlify

Required Tools:
. Code Editor: Visual Studio Code (or your preferred editor)
. Browser: Google Chrome (or any modern browser)
. Node.js & npm: To manage dependencies and run the React
application
